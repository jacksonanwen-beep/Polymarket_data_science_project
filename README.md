# **The Illusion of the Safe Bet in Polymarket Election Events**

## **Project Overview**
This project analyses the accuracy of Polymarket predictions in relation to elections by scraping historical election prediction data, calculating the market's mathematical error rates (Brier Scores), and using machine learning to hunt for systemic flaws.

## **Repository Structure**
`data_gathering.py` – Script to extract data from Polymarket Gamma and CLOB APIs
 `flattening_json.py` - Script to parse and clean the raw Polymarket JSON data
`analysis.py` - Contains the script for a calibration curve, histogram, OLS regression(s), coefficient plot and a causal forest model
`/images/` - Contains the plots generated by the models
`first_regression_results.txt` - The raw statistical output and P-values from the first OLS summary
`second_regression_results.txt` - The raw statistical output and P-values from the second OLS summary

## **How to Run**
1. Clone this repository
2. Activate your virtual environment: ‘source .venv/bin/activate’ (Mac/Linux) or ‘.venv\Scripts\activate’ (Windows).
3. Install the required packages:
```pip install pandas numpy matplotlib seaborn statsmodels scikit-learn econml```
4. Run the files in the correct order (data_gathering.py --> flattening_json.py --> analysis.py)
